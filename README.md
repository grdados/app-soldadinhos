# Soldadinhos de Jesus - Landing Page

Landing page institucional do projeto social **Soldadinhos de Jesus**, com foco em:

- contribuição e voluntariado;
- apresentação do projeto;
- mural de encontros;
- visão da área de soldadinhos (pontuação, carteirinha e estrelinhas);
- chamada para blog e localização.

## Estrutura

- `index.html`: conteúdo da landing page
- `styles.css`: estilos visuais e responsividade
- `script.js`: menu mobile e ano automático no rodapé

## Como visualizar

Abra o arquivo `index.html` no navegador.

## Próximos passos recomendados

1. Substituir imagens placeholder por fotos oficiais do projeto.
2. Conectar botões de contribuição e WhatsApp com links reais.
3. Migrar para Next.js para integrar blog e painel admin na próxima etapa.
